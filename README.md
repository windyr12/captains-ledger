

# Skipper's Ledger

A plugin for tracking community trawling trips. Essentially a taximeter for ironmen.

## Features

- Track crew members using the net by detecting players standing on the same tile as you.
- Show the active crew count, up to a maximum of 9 players.
- Track depositing and banking players separately.
- Automatically detect ironmen and set them as banking. This can be manually overridden. It does not track group ironmen.
- Calculate GP owed from a configurable hourly rate.
- Track total GP earned during the trip.
- Track paid/unpaid status for each player.
- Copy a Discord-friendly trip summary to clipboard.
- Reset tracked players, ignored players, and payments when needed.
